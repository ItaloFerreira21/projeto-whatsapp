# Projeto WhatsApp - Frontend

Este é o frontend do projeto WhatsApp, um sistema de autenticação de usuários usando React, TypeScript, Axios e JWT.

## Requisitos

- Node.js (versão 14 ou superior)
- npm ou yarn
